# 👑 Conquest IO: Total Domination

[![Live Demo](https://img.shields.io/badge/🚀_Live_Demo-Start_Here-3b82f6?style=for-the-badge&logo=github&logoColor=white)](https://nirgranthi.github.io/Conquerer_Ultimate/)

**Conquest IO** is a high-stakes, massive-scale abstract strategy game where over 10 civilizations battle for supremacy on a single map. It's a "Total Domination Simulator" designed for strategy enthusiasts who crave pure, distraction-free warfare.

Hold territory to grow your population. Launch synchronized attacks to seize new lands and outmaneuver your enemies. Simple to learn, but brutal to master.

---

## 🔥 Key Features

*   **🔗 Multi-Base Chain Attacks**: Coordinate massive assaults by dragging through your allied nodes to chain-select them for a single, overwhelming target.
*   **⚡ Global Assault (The Nuke)**: Double-tap any node to command **50% of troops from ALL your bases** to strike that target instantly.
*   **🤖 Adaptive AI**: Scaling levels from **Easy** for a relaxed conquest to **Impossible** for those who want to face a ruthless strategist.
*   **💥 Real-Time Collisions**: Tactical unit physics mean troops die on collision mid-path. Timing and path planning are your greatest weapons.
*   **📱 Unified Experience**: Fully optimized for both Desktop (Mouse) and Mobile (Touch/APK).

---

## 🎮 How to Play

1.  **Attack**: Drag from your base to an enemy or neutral node.
2.  **Chain**: Drag through multiple allied bases to select them all at once.
3.  **Nuke**: Double-tap a target node to launch a Global Assault from all bases.
4.  **Win**: Capture every node on the map to achieve Total Domination!

---

## 🛠️ Tech Stack

Built with a cutting-edge frontend stack for maximum performance and cross-platform compatibility:

*   **React 19 + TypeScript**: Powering a responsive and modern UI.
*   **Tauri 2**: Seamlessly delivering the game as a native desktop and mobile application.
*   **Three.js / OGL**: Driving the stunning "Hyperspeed" and "Galaxy" visual effects.
*   **Vite**: For lightning-fast development and optimized production builds.
*   **Tailwind CSS 4**: A sleek, custom design system.

## 🚀 Getting Started

### Prerequisites
*   Node.js (LTS version)
*   npm or yarn

### Installation
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Build for Desktop (Tauri):
   ```bash
   npm run tauri build
   ```
